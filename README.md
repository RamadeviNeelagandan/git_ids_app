# 🔐 GAN-Based Intrusion Detection System (IDS)

## 📌 Project Description

This project is a **GAN-based Intrusion Detection System (IDS)** that detects malicious network traffic and classifies it as **Benign** or **Attack** using deep learning techniques.

---

## 🚀 Features

* GAN-based anomaly detection
* Real-time prediction using Streamlit UI
* Interactive visualization
* User-friendly interface

---

## 🛠️ Tech Stack

* Python
* PyTorch
* Streamlit
* NumPy
* Matplotlib

---

## 📥 Setup Instructions

### Step 1: Clone the repository

git clone https://github.com/YOUR-USERNAME/git_ids_app.git
cd git_ids_app

---

### Step 2: Install dependencies

pip install -r requirements.txt

---

### Step 3: Add required files

Download the dataset and model files from the Google Drive link and place them in the project folder.

---

### Step 4: Run the application

python -m streamlit run gans_ids_app.py

---

## 🌐 Output

The app will open in your browser at:
http://localhost:8501
